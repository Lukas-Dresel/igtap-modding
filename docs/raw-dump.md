# Raw Data Dump

Complete typetree dump of every object from every Unity file.
Generated by `extract_all_objects.py`. Located in `output/raw_dump/`.

**157 files, 456 MB total.**

Files are named `{source_file}__{object_type}.json`. Each file is a JSON array
of objects, where each object has:
- `path_id`: unique object identifier within the file
- `type`: Unity object type name
- `byte_size`: serialized data size
- `typetree`: full typetree dict with every serialized field
- `m_Name`: object name (if applicable)
- `raw_size`: raw byte count

## Usage

The raw dump is the single source of truth. All other extracted data files
(layer JSONs, scene objects, sprite indices) are derived from this data.

To query specific objects:
```python
import json
with open('output/raw_dump/level1__SpriteRenderer.json') as f:
    renderers = json.load(f)
for r in renderers:
    tt = r.get('typetree', {})
    print(r.get('m_Name'), tt.get('m_DrawMode'), tt.get('m_Size'))
```
